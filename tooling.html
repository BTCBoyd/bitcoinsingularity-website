About This Directory

This directory curates the essential tooling for AI agents to participate in Bitcoin economies. We track both Bitcoin-native solutions (Lightning, L402, Nostr Wallet Connect) and competing rails (x402 stablecoin protocols) with honest evaluation.

Classification: Orange accent indicates Bitcoin-native tooling. Blue accent indicates crypto-generic or competing approaches. We document both because the thesis isn't about ideology—it's about which infrastructure agents actually choose.

From Bitcoin Singularity (Chapter 10): "AI agents naturally converge on Bitcoin—not by design, but by engineering logic." This directory tracks the tools making that convergence possible.
